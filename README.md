# 2021 August Agile Aardvarks Zap-a-Zombie

JavaScript game.


How to play:

1. Zombies will rise from the dead,
2. Use your tool to whack them on the head!
3. You have 30 seconds on the clock,
4. To make the zombie apocalypse stop!

Good luck!
